# Tap To Do
### A simple to do list app focused on adding and completing task quickly.
**One tap** to start typing

**One tap** to assign a simple due date

**One swipe** to complete

#

Tap To Do was designed to make adding a task as frictionless as possible. The focus is on productivity and completion, *not list managment.*
- Quick task entry
- Simple gesture navigation
- Automatic Sorting
- Themes
- More context can be added to a task by tapping it
  - Notes
  - Images
  - Advanced Due Date/Time
- Completed tasks stay visible to highlight daily progress
- Completed tasks move to the bottom after a day
- Completed tasks are automatically cleaned after a week
